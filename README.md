## About

PHALENDROME SOLUTIONS provides website design, digital platform development, and IT consulting services. This repository contains the company's public-facing website.

## Tech Stack

- **HTML5** — semantic, accessible markup
- **CSS3** — custom properties, responsive design, CSS Grid/Flexbox
- **Bootstrap 5.3** — responsive navigation and layout utilities
- **Vanilla JavaScript** — form validation, scroll animations
- **Google Fonts** — Inter typeface

## Setup

There's no build step required. Open any `.html` file in a browser or serve locally:

```bash
npx serve
# or
npx live-server
```

## Deployment

The site is deployed via **GitHub Pages** from the `main` branch root.

**Live URL:** https://devphalendrome.github.io/PHALENDROME/

To deploy updates:
```bash
git add -A
git commit -m "description of changes"
git push origin main
```

GitHub Actions automatically builds and deploys to Pages.
