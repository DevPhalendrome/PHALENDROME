# PHALENDROME SOLUTIONS - Logo Assets

## Available Logo Formats

### 1. **logo.svg** (Original / Web Default)
- **Dimensions:** 240×48px
- **Best for:** Website header, favicon conversion
- **Format:** SVG (scalable)
- **Usage:** Default navbar logo

### 2. **logo-high-res.svg** (Landscape - High Resolution)
- **Dimensions:** 1200×240px
- **Best for:** Presentations, large displays, print materials
- **Format:** SVG (infinitely scalable)
- **Aspect Ratio:** 5:1 (wide landscape)
- **Color Scheme:** Modern blue (#2563EB primary, #1e40af secondary)

### 3. **logo-square.svg** (Square Format)
- **Dimensions:** 400×400px
- **Best for:** Social media profiles, avatars, app icons, favicon
- **Format:** SVG (infinitely scalable)
- **Aspect Ratio:** 1:1 (square)
- **Color Scheme:** Modern blue (#2563EB primary, #1e40af secondary)

## Export Instructions

To export any SVG to PNG or other formats at high resolution:

### Option A: Using Online Tools
1. Visit https://cloudconvert.com/ or https://ezgif.com/
2. Upload the SVG file
3. Choose output format (PNG, JPG, PDF)
4. Set resolution (e.g., 300 DPI for print, 72 DPI for web)
5. Download

### Option B: Using Adobe Illustrator
1. Open SVG file
2. File > Export As
3. Choose PNG or AI format
4. Set resolution to 300+ DPI
5. Save

### Option C: Using Command Line (ImageMagick)
```bash
convert -density 300 logo-high-res.svg -quality 100 logo-high-res.png
```

## Color Palette

- **Primary Blue:** #2563EB (rgb(37, 99, 235))
- **Secondary Blue:** #1e40af (rgb(30, 64, 175))
- **Design:** Symmetric geometric marks + modern typography

## Usage Guidelines

- **Minimum size (web):** 120px width
- **Minimum size (print):** 1 inch width at 300 DPI
- **Clear space:** Maintain 20% space around logo
- **Never distort** the aspect ratio
- **Use full color** on light backgrounds
- **Use white** on dark backgrounds (invert colors as needed)
